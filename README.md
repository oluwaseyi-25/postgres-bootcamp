# Coach Ola's 12-Week PostgreSQL Bootcamp

A comprehensive bootcamp covering PostgreSQL fundamentals and advanced topics over 12 weeks.

## Contents

This repository contains homework assignments and SQL exercises from the bootcamp:

- [day_2_homework.sql](day_2_homework.sql) - Week 2 SQL exercises
- [day_4_homework.sql](day_4_homework.sql) - Week 4 SQL exercises

## About

This bootcamp covers essential PostgreSQL skills including:
- Database design and normalization
- SQL queries and optimization
- Indexing and performance tuning
- Advanced PostgreSQL features

## Getting Started

To run the SQL files, you'll need PostgreSQL installed. Execute the files in your PostgreSQL environment using:

```sql
\i day_2_homework.sql
\i day_4_homework.sql
```

For more information and additional materials, refer to the weekly assignments as you progress through the bootcamp.
